# Automations

Protocols and prompt templates only. This directory intentionally contains no
live scheduled records, target thread IDs, cron jobs, launchd jobs, daemons,
shell watchers, sends, publishes, pushes, remotes, or third-party effects.

Default templates:

- `morning-brief.md`: review-only daily brief protocol
- `dream-cycle.md`: review-only weekly dreaming/connection protocol
- `weekly-review.md`: review-only weekly project and open-loop review
- `project-refresh.md`: manual project-memory closeout protocol
- `codex-automation-specs.md`: copyable Codex heartbeat prompts

Prefer Yerhed MCP with file fallback. Use dedicated Codex heartbeat threads for
recurring loops, and keep every loop review-only for external side effects.
